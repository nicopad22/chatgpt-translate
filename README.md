I've started working on this to help my mom with translations, using chatGPT to create more fluent and context-aware translations from english to spanish and otherwise. Of course, other languages could be added in the future.

I have code that works with excel, word and powerpoint files, and am currently working in a pdf file, although without much luck. Trying to create a repository to formalize the project somewhat, and to have some history and to be able to trace back changes.

You need to create a "TOKEN.txt" file in the project folder and put your openai key in it for it to work. Make sure its in the .gitignore!!!!
